Tentang Lab

SMK BN 666 memiliki laboratorium komputer untuk menunjang kegiatan belajar mengajar siswa. Laboratorium "Lab-02" ini dilengkapi dengan 20 buah komputer desktop dan 20 laptop. Siswa bisa menggunakan fasilitas komputer maupun laptop tersebut untuk kegiatan praktikum siswa maupun yang lainnya. Komputer desktop sudah dilengkapi dengan berbagai software pemrograman seperti: Visual Studio Code, XAMPP, Androd Studio, Netbeans, dan lain-lain

Siswa RPL bisa menggunakan software Visual Studio Code tersebut untuk belajar pemrograman web. Lab komputer ini berada di lantai 1, bersebelahan dengan "Lab-01". Komputer-komputer tersebut sudah terhubung ke internet. Siswa dan guru bisa memanfaatkan fasilitas komputer yang ada di laboratorium komputer ini untuk mengerjakan tugas maupun yang lainnya. Di laboratorium komputer ini juga dilengkapi dengan proyektor untuk memudahkan guru ketika mengajar siswa.
